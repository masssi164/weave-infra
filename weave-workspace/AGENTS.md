# Workspace Guide

`weave-workspace/` is the runnable local stack workspace.

## Files

- `install.sh`: applies both Terraform stages, waits for service readiness, and bootstraps Nextcloud OIDC.
- `.generated/bootstrap.env`: git-ignored local bootstrap inputs generated by `install.sh` for idempotent reruns.
- `01-infrastructure/`: Docker runtime stage.
- `02-keycloak-setup/`: Keycloak configuration stage.

## Maintenance Notes

- Keep stage root modules thin and orchestration-focused.
- Put tightly coupled resources inside the existing child modules under each stage’s `modules/` directory.
- Update the corresponding stage `AGENTS.md` when files or responsibilities move.
